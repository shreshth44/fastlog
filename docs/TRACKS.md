# Project Tracks

## Track A: I/O + Parsing
- **Scope**: Reading large files, parsing lines into `LogEntry`.

## Track B: Aggregation + Stats
- **Scope**: Computing counts, averages, top-k.
- **Deliverables**: `StatsEngine` class.

## Track C: Performance
- **Scope**: Benchmarking implementation.
- **Deliverables**: Reproducible perf reports.

## Track D: Optimization
- **Scope**: Thread pool, custom allocators.
- **Deliverables**: 4x speedup.
